Social Media Application (Console Based)

📌 Project Description

This project is a Console-Based Social Media Application implemented in Java using Low Level Design (LLD) principles.
It allows users to sign up, send friend requests, create posts, view feeds, and interact with posts through likes and comments.

The system is structured using a layered architecture with packages such as bean, service, util, and main to maintain modularity and clean code design.

🚀 Features

1️⃣ User Management

User Signup

Store user details:

Name

Date of Birth

Age

Location

Occupation

2️⃣ Friend Request Management

Send Friend Requests

View Pending Requests

Accept Friend Requests

Maintain Friend List

3️⃣ Post Management

Create Posts

Posts stored for each user

4️⃣ News Feed

View posts from friends

Latest posts appear in feed

5️⃣ Post Interaction

Like Posts

Comment on Posts

View Like Count and Comments

6️⃣ Notification System

Notifications for:

Friend Requests

Post Interactions

Shared Posts
